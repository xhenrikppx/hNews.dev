Caption

Add captions with different animations and backgrounds to images or Slideshow items.

Usage

To apply the Caption component to an image, wrap the image in a container element and add the .uk-caption-toggle class. Add the .uk-caption class to another element inside the container to create the caption itself.

You can also add an animation from the Animation component to the caption. To trigger the animation you also need to add the .uk-animation-hover class to the element with the caption toggle.

Example

Markup

<figure class="uk-caption-toggle uk-animation-hover">
   <img src="" alt="">
   <figcaption class="uk-caption uk-animation-fade">...</figcaption>
</figure>

Note To see a slideshow with captions, go to the Slideshow component.

Caption panel

To give the caption a background just add the .uk-caption-panel class.

Example

Markup

<figure class="uk-caption-toggle">
   <img src="" alt="">
   <figcaption class="uk-caption uk-caption-panel">...</figcaption>
</figure>

Caption position

To position the caption, add one of the following classes.

Class Description
.uk-caption-top Add this class to align the caption to the top.
.uk-caption-bottom Add this class to align the caption to the bottom.
.uk-caption-left Add this class to align the caption to the left. You can also add one of the .uk-width-* classes from the Grid component.
.uk-caption-right Add this class to align the caption to the right. You can also add one of the .uk-width-* classes from the Grid component.

Example

Markup

<figure class="uk-caption-toggle">
    <img src="" alt="">
    <figcaption class="uk-caption uk-caption-right uk-width-1-2">...</figcaption>
</figure>

Caption and flex

You can use the Flex component to center captions vertically and horizontally.

Example

Markup

<figure class="uk-caption-toggle">
    <img src="" alt="">
    <figcaption class="uk-caption uk-flex uk-flex-center uk-flex-middle uk-text-center">
        <div>...</div>
    </figcaption>
</figure>
